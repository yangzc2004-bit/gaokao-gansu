# 甘肃高考志愿推荐系统

面向甘肃新高考场景，重点挖掘专项计划与政策红利。

## 产品目标

- 左侧录入用户画像：历史/物理、四选二、分数、位次、地区、户籍、民族、学籍等
- 自动识别可报专项计划与政策红利
- 右侧输出四类志愿：冲、稳、保、政策红利
- 每个志愿卡片展示：院校信息、历年位次趋势图、CV 波动图、历年平均位次、推荐理由

## 当前范围

- 省份：甘肃
- 推荐主轴：位次优先
- 科类入口：先选历史/物理，再选四选二
- 后端：Python
- 前端：Streamlit

## 目录结构

- `docs/`：架构、数据、规则设计
- `data/`：原始数据、中间层、清洗结果
- `backend/`：数据清洗、规则引擎、推荐引擎、API
- `frontend/`：Streamlit 页面与组件
- `configs/`：规则 JSON、区域字典、字段映射

## 开发阶段

1. 数据清洗与标准化
2. 规则结构化与资格判定
3. 位次预测与冲稳保分层
4. 政策红利推荐
5. Streamlit 前端与可视化
6. 导出与上线

## 注意

当前系统主打 `位次优先 + 可解释推荐`，生成推荐时建议必须填写全省位次，并选择两门再选科目。
